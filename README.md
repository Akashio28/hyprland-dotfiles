## 📦 Contents

| Config | Description |
|--------|-------------|
| **Hyprland** | Window manager |
| **Waybar** | Status bar |
| **Swaync** | Notification center |
| **Rofi / Wofi / Tofi** | App launchers |
| **Kitty** | Terminal emulator |
| **Cava** | Audio visualizer |
| **Btop** | System monitor |
| **Fastfetch** | System info |
| **Wallust / Wal** | Color scheme generator |
| **GTK 3 & 4** | Theme config |
| **Wlogout** | Logout menu |
| **Starship** | Shell prompt |

---

## ⚠️ Requirements

Make sure you have the following packages installed before running the install script:

```bash
sudo apt install hyprland waybar kitty rofi wofi git
```

---

## 🚀 Installation

> ⚠️ Your old configs will be automatically backed up to `~/.config/name.bak`

```bash
git clone https://github.com/Akashio28/hyprland-dotfiles.git && cd hyprland-dotfiles && bash install.sh
```

---

## 🔄 Update

To get the latest changes:

```bash
cd hyprland-dotfiles
git pull
bash install.sh
```
